The Tasking Manager 3 has new features and functionality that are meant to improve the experience of everyone that uses it. Most of the individual improvements bring benefits to each group of users, mappers, validators and project managers. But the improvements can be divided into three broad categories: Improvements for Mappers, improvements for validators and improvements for mapping project managers.

Improvements for Mappers

  • Expanded User Profile and Contact information - Please make the first thing you do in the new Tasking Manager visiting your {{ 'user profile' | translate }}. There you will see your mapper experience level and most importantly, can fill in your user contact information so you can get feedback on your mapping delivered to your email in-box.

  • Finding a Project - When you visit the Contribute link to find a project that needs mapping, you will have a new set of options to help you find a project to work on. You can now filter by:

    • Difficulty - Every new project will have a difficulty rating so you can work on projects that are best for your experience level.

    • Organization - Select from the list of organizations that make use of OpenStreetMap map data

    • Campaign - Often projects are grouped into campaigns, efforts that need multiple projects to get done, now you can easily find all the projects in a campaign.

    • Type of Mapping - Roads, buildings, waterways, find the kind of mapping you are in the mood for.

  • More ways to ask questions and get feedback - There is now a "Questions and Comments" tab on each project, with a quick link to direct a question to the mapping project manager.

Improvements for Validators

  • Expanded User Profile and Contact information - Please make the first thing you do in the new Tasking Manager visiting your {{ 'user profile' | translate }}. The Tasking Manager now tracks and displays the number of Tasks you have validated. Since users can also enter their email addresses, when you direct mapping feedback to a user, they are much more likely to see your feedback (make sure to add your email address to your profile too.)

  • Validate multiple tasks at once - While you can still validate one task at a time, you can also checkout multiple tasks at once to validate. You have to option of drawing a polygon around a group of tasks, or you can select all the tasks completed by an individual mapper. There is a dedicated "Validation" tab with a list of all the contributors to a project and some key information to help you priortize your validation. You can sort contributing users by:

    • Experience level - Based on the number of changesets a user has in OSM you can get an immediate idea of how experienced a mapper is.

    • Registration date - This tells you how long a user has been using the Tasking Manager. It will help you find brand new users who can benefit from immediate feedback.

    • Time since last validation - This will let you know how long it has been since a user received feedback.

  • Project level comments and full task comment feed - With the new "Questions and Comments" tab on each project and a full list of all the comments on individual tasks, you will be able to find users who have questions or identify issues with the project quickly.
